# Реестр задач внедрения — детальная спецификация

Подробный нумерованный backlog по промпту оператора (ТЗ 2026-06-25). **Последовательность не
фиксирована** — порядок решаем отдельно. Стратегия/фазы — в [`IMPLEMENTATION_PLAN.md`](IMPLEMENTATION_PLAN.md).

Каждая задача расписана по единому шаблону:
- **Цель** — что получает пользователь.
- **Подзадачи** — конкретные шаги реализации (адресуемые: `N.1`, `N.2`…).
- **API / сигнатуры** — предлагаемые функции/классы (помечено «предлагается»; источник-сигнатуры из
  SpectraVibe помечен «источник»).
- **Параметры / формулы** — точные значения, с провенансом `file:line` где портируем.
- **UI** — конкретные элементы интерфейса.
- **Тесты** — проверяемые критерии готовности.
- **Зависимости** — пакеты и задачи-предпосылки.
- **Объём** — S (≤полдня) / M (1–2 дня) / L (3+ дня), грубо.

Сложность данных: 🟢 чистый numpy/stdlib · 🟡 нужен scipy/sklearn · 🔵 только UI · 🟣 требует рефактора
модели. SpectraVibe (`gamma-spectrum-analysis`) — **референс на чтение**, пути `scripts/gamma/...`.

Сквозной инвариант реализации: код ≥25 строк → спецификация `scripts/ollama/_spec_<name>.md` →
кодоген Ollama `qwen3-coder:30b` → валидация (`py_compile`+pytest). Численное ядро — без Qt, в
`awf/analysis/` или `awf/model/`. Калибровка — ascending-полином (совместима с SpectraVibe).

## Задача 3 — Континуум/фон под пиком 🟢 · Объём M
**Цель:** оценить baseline под ROI пика для расчёта нетто.

**Подзадачи:**
- 3.1. Cowell: полином 1-го порядка по крыльям ROI (`numpy.polyfit`), интеграл baseline по full ROI.
- 3.2. Сглаженная ступень (Будыка 7.29): `B(E)=bg_r+(bg_l−bg_r)·0.5·erfc((E−E₀)/(σ√2))+slope·(E−E₀)`,
  `erfc` через `math.erfc` (без scipy).
- 3.3. Асимметричные окна фона: автосужение крыла при интерферирующем соседнем пике.
- 3.4. Выбор модели — параметр `method ∈ {'cowell','step'}`.

**API (предлагается):**
```python

                       wing=None) -> np.ndarray   # baseline по каналам ROI
```
**Источник:** `peaks/area.py:77–238` `cowell_area()`; `peaks/background_options.py:40–89`
`smoothed_step_bg()`; асимметрия `background_options.py:107–169`.
**Тесты:** на чисто-линейном фоне Cowell даёт нетто≈0; площадь известной гауссианы поверх фона в

## Задача 7 — MDA по ISO 11929 🟢 · Объём S
